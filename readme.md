# ✨ 1. 파이썬용 한국투자증권 웹 서비스 소개

한국투자증권의 트레이딩 OPEN API 서비스를 파이썬 환경에서 사용할 수 있도록 만든 강력한 운용 라이브러리를 이용한 자동투자 웹서비스입니다.

## 1.1. 웹 서비스 특징
<details>
서버에서 자동으로 투자하고 해당 이력을 보여줍니다.
</details>

<details>
인증키를 입력하면 1일 단위 주가 조회 후 매매/매도를 합니다.
</details>
